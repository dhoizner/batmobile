brake 8D

drive scheme 2

drive scheme 3

throttle Ch3

drive scheme 4

------------------------

remote control - 7L or 7D
onboard control - 7U
onboard control - 5U or 5D held

boost - 6U
drift - 6D

digital output 1 - 8U (toggle)
digital output 2 - 8L (toggle)

RSL
blinking - kid mode
solid - parent mode
uncontrolled = off
